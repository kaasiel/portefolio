📋 Présentation du Projet

Nom du projet : Schwa

Description : Schwa est une plateforme web interactive dédiée à l'apprentissage de la langue anglaise. Le nom "Schwa" fait référence au son vocalique neutre (ə) le plus fréquent en anglais, symbolisant l'importance de la prononciation et de la maîtrise des fondamentaux de la langue.

Date de création : Décembre 2025

🎯 Objectifs du Projet

Schwa vise à offrir une expérience d'apprentissage complète de l'anglais à travers quatre piliers essentiels :

  • Grammar - Maîtrise des règles grammaticales et structures de phrases
  • Vocabulary - Enrichissement du vocabulaire par niveaux (A1 à C2)
  • Listening - Développement de la compréhension orale
  • Reading - Amélioration de la fluidité et compréhension écrite

🛠️ Technologies Utilisées

Frontend

  • HTML5, CSS3 (fichiers modulaires : main2.css, grammar.css, darkmode.css, ...)
  • JavaScript (Vanilla) — architecture orientée objet pour les leçons, gestion d'état via localStorage
  • Timer et tracking de progression côté client

Backend

  • PHP pour le traitement serveur, sessions et API REST (ex : save_score.php)
  • MySQL pour la persistance
  • Connexion via router/Dbconnection.php

Architecture

Architecture MVC simplifiée — séparation Landing (public) & Application (app). Système de modules avec includes PHP (header, main, footer).

🎓 Module Vocabulary - Architecture

Classe VocabularyLesson (JS) :

class VocabularyLesson {
  constructor(level, title, description, category, introduction, wordsList, usageTips)
  render() // Affiche la leçon à l'écran
}

Chaque mot contient : word, type, translation, example, exampleTranslation, usageTips.

Leçons disponibles : listes par niveau (A1 à C2), exemples A1 : Greetings, Numbers, Family, Colors, Food, Animals, Days & Months.

🧪 Système d'Exercices

  • 10 questions QCM par leçon
  • +10 points par bonne réponse (score max 100)
  • Seuil de validation : 65%
  • Timer démarré au début et sauvegardé

Format de question (exemple) :

{
  question: "Texte de la question",
  options: ["Option 1","Option 2","Option 3","Option 4"],
  correct: 1,
  explanation: "Explication détaillée"
}

🔧 Fonctionnalités Principales

  • Système d'onglets Leçon / Exercices
  • Filtrage par niveau (A1..C1)
  • Cartes de mots avec exemples
  • Mode sombre (persisté)
  • Feedback visuel immédiat et statistiques en temps réel

📊 API REST — save_score.php

Endpoint : POST /save_score.php

Headers : Content-Type: application/json

Body JSON attendu :
